# BRDF 的性质

线性 ： [1：07：12] 可折分再叠加
可逆性: \\(fr(wi -> wo) = fr(wo -> wr) \\)，即交换入射方向与出射方向，BRDF不变
能量守恒：\\(\int_{H^2}fr(wi -> wo)\cos\theta_idw_i \le 1\\)，总能量不可能变多，但有可能被吸收
各向同性：[1:09:40] \\(fr(\theta_i, \phi_i;\theta_r, \phi_r) = fr(\theta_i, \phi_i, \phi_r - \phi_i)\\)，其中\\(\theta_r\\), \\(\phi_r\\为方位角)
